# Dependencies
### Backend:
- prompt-sync
- bcrypt
- mongoose
- cors
- express
- dotenv

### Frontend:
- cypress
- jest
- tailwind
- postcss
- autoprefixer
- react-router-dom

## Backend:
Inside the backend Directory, run <br>
`npm install prompt-sync bcrypt mongoose cors express dotenv`

## Frontend
Inside the frontend Directory, run <br>
`npm install` <br>
`npm install react-router-dom`

## Launching React on port 3000
`npm start`
