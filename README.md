# Dependencies
### Backend:
- prompt-sync
- bcrypt
- mongoose
- cors
- express

### Frontend:
- cypress
- jest
- tailwind
- postcss
- autoprefixer

## Backend:
Inside the backend Directory, run <br>
`npm install prompt-sync bcrypt mongoose cors express`

## Frontend
Inside the frontend Directory, run <br>
`npm install`

## Launching React on port 3000
`npm start`
