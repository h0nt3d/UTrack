# Dependencies
### Backend:
- prompt-sync
- bcrypt
- mongoose
- cors
- express
- dotenv

### Frontend:
- cypress
- jest
- tailwind
- postcss
- autoprefixer
- react-router-dom
- xlsx

## Insatlling Dependencies
- For Windows, inside UTrack Folder, run <br>
`INSTALL_DEPENDENCIES.bat`
- For Linux and MacOS, run <br>
`INSTALL_DEPENDENCIES.sh`

## Launching React on port 3000
`npm start`
